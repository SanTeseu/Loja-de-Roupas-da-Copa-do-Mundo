# Arquitetura: E-commerce Copa do Mundo

## Visão Geral
Sistema de e-commerce focado na venda de camisas oficiais das seleções da Copa do Mundo. O sistema deve suportar alto tráfego e garantir a integridade do estoque em tempo real.

## Tech Stack
- **Frontend:** Next.js 14 (App Router), React, TailwindCSS.
- **Backend:** Node.js com Express e TypeScript.
- **Banco de Dados:** PostgreSQL.
- **ORM:** Prisma.

## Estrutura de Diretórios Backend
- `/src/controllers`
- `/src/services`
- `/src/routes`
- `/prisma/schema.prisma`
